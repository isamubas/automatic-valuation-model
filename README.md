# automated valuation model
-this model is supposed to predict the prices of  a house, given different features like bedrooms,bathrooms,house_size, and among others, whether mansion or naormal home, it has an accuracy of 82% Rsquared score
-it was trained on data of over 2.2 mill rows, and 12 features, its brain is quiet heavy to load even after compressing it.
- i am working on a better version so far has an accuracy of 83%, and using streamlit, you will be able to input the features of the house and get a price, while its putting into consideration the market analysis to predict the price for you.
